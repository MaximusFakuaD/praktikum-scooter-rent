# Практический блок: вторая часть финального проекта

## Яндекс Самокат — Тестирование веб-приложения и API

Вторая часть практической работы включает:  
- Работа с базой данных PostgreSQL (запросы к таблицам Couriers и Orders)  
- Автоматизированный тест API (создание заказа + получение по треку)

## Структура репозитория

- `db-queries/` — работа с базой данных  
  - `database_work.md` — отчёт с запросами и результатами  
  - `screenshots/` — скриншоты подключения и вывода запросов  

- `api-automation/` — автоматизация теста API  
  - `test_api.py` — скрипт теста  
  - `README.md` — подробное описание автотеста  
  - `screenshots/` — скрин успешного запуска теста  

## Окружение
- ОС: Windows 11  
- Инструменты: Git Bash, VS Code, Python 3, psql (PostgreSQL client), requests  
- Доступ к БД: через SSH к серверу тренажёра + psql локально на сервере  
- Доступ к API: тестовый стенд {актуальный_id}.serverhub.praktikum-services.ru

## Результаты
- База данных: успешно подключился, выполнил запросы. Изначально таблицы были пустыми, после создания заказов появились данные.  
- Автотест API: успешно создаёт заказ и получает его по треку (статус 200).
