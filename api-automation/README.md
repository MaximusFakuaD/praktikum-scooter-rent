# Автотест API Яндекс Самокат

## Описание
Автотест проверяет сценарий:
1. Создание заказа через POST /api/v1/orders
2. Получение заказа по треку через GET /api/v1/orders/track?t={track}
3. Проверка статуса ответа = 200

## Требования
- Python 3.8+
- Библиотека requests

## Запуск
```bash
python test_api.py
