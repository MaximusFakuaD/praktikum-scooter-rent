# Автотест API Яндекс Самокат  
Финальный проект. Корнеев Леонид инженер по тестированию плюс — 40-я когорта  

## Описание проекта

Проект содержит автотесты API сервиса Яндекс Самокат (endpoint'ы заказов).

Проверяемый позитивный сценарий:
- Создание заказа (POST `/api/v1/orders`)
- Получение созданного заказа по трек-номеру (GET `/api/v1/orders/track?t={track}`)
- Проверка корректности возвращённого трек-номера и статуса ответа

## Структура проекта

api-automation/
├── configuration.py          # базовый URL и пути эндпоинтов
├── data.py                   # заголовки и тело запроса
├── sender_stand_request.py   # функции-обёртки для HTTP-запросов
└── test_order_creation.py    # файл с автотестами (pytest)
