# Автотест API Яндекс Самокат  
Корнеев Леонид, 40-я когорта — Финальный проект. Инженер по тестированию плюс

## Описание проекта

Проект содержит автотесты API сервиса Яндекс Самокат

Проверяемый позитивный сценарий:
- Создание заказа (POST `/api/v1/orders`)
- Получение созданного заказа по трек-номеру (GET `/api/v1/orders/track?t={track}`)
- Проверка корректности возвращённого трек-номера и статуса ответа

## Структура проекта

api-automation/
- `configuration.py` - базовый URL и пути эндпоинтов
- `data.py` - заголовки и тело запроса
- `sender_stand_request.py` - функции для HTTP-запросов
- `test_order_creation.py` - файл с автотестами (pytest)

- `README.md` - файл с описанием проекта  
- `.gitignore` - игнорирование служебных и ненужных файлов 

## Перед запуском
В файле configuration.py заменить ссылку в URL_SERVICE на актуальную

## Требования

- Python 3.8+
- Библиотеки:
    - requests
    - pytest
